# Bash Scripting – Fondamentali (Variabili, Condizioni, Loop, Funzioni)

Per imparare ed approfondire bash scripting è consigliata la seguente risorsa:
[https://devhints.io/bash](https://devhints.io/bash). 


## Cheatsheet

### 1. Struttura base

```bash
#!/bin/bash

# Comment
echo "Hello World"
````

Renderlo eseguibile

```bash
chmod +x script.sh
./script.sh
```

### 2. Variabili

```bash
name="Alex"
echo $name
echo "${name}"
```

Leggere input utente

```bash
read -p "Enter your name: " name
```

### 3. Positional Parameters (quando viene avviato lo script)

```bash
echo $0   # Script name
echo $1   # First argument
echo $2   # Second argument
echo $#   # Number of arguments
echo $@   # All arguments
```

### 4. Condizioni

```bash
if [ "$name" == "Alex" ]; then
  echo "Hello Alex"
elif [ "$name" == "Bob" ]; then
  echo "Hello Bob"
else
  echo "Unknown user"
fi
```

File checks:

```bash
[ -f file.txt ]   # File exists
[ -d dir ]        # Directory exists
[ -z "$var" ]     # Variable is empty
```

### 5. Cicli (loops)

#### Ciclo for

```bash
for i in 1 2 3; do
  echo $i
done
```

#### Ciclo for su file

```bash
for file in *.txt; do
  echo "$file"
done
```

#### While loop

```bash
count=1
while [ $count -le 5 ]; do
  echo $count
  ((count++))
done
```

### 6. Funzioni

```bash
greet() {
  echo "Hello $1"
}

greet "Alex"
```

### 7. Codici uscita

```bash
exit 0   # OK
exit 1   # Errore
```

Controllo ultimo comando

```bash
echo $?  # Exit status
```


### 8. Operatori utili

```bash
&&   # Comando successio se ha successo
||   # Comando successivo anche in caso di fallimento
|    # Pipe output (Controllare cosa cambia tra Pipe, Redirect e Append)
>    # Redirect output
>>   # Append output
```

Esempio:

```bash
grep "error" log.txt | wc -l
```

### 9. Built-ins usati comunemente

```bash
echo
read
cd
pwd
export
alias
```
