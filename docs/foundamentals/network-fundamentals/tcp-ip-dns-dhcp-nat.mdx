# TCP/IP, DNS, FQDN, DHCP, NAT

Le reti informatiche moderne si basano su protocolli e servizi che consentono la comunicazione tra dispositivi. Tra i più importanti troviamo TCP/IP, DNS, FQDN, DHCP e NAT:

* **TCP/IP**: è l’insieme di protocolli su cui si basa Internet. IP si occupa dell’indirizzamento e dell’instradamento dei pacchetti di dati, mentre TCP garantisce che i dati arrivino correttamente e nell’ordine giusto tra mittente e destinatario.

* **DNS (Domain Name System)**: serve a tradurre i nomi di dominio (come *[www.google.com](http://www.google.com)*) in indirizzi IP numerici, permettendo ai dispositivi di trovare i server corretti sulla rete.

* **FQDN (Fully Qualified Domain Name):** è il nome di dominio completo e assoluto, come `www.example.com`, che specifica la posizione esatta di un host all'interno della gerarchia DNS, garantendo un'identificazione univoca.

* **DHCP (Dynamic Host Configuration Protocol)**: assegna automaticamente agli host della rete parametri di configurazione come indirizzo IP, gateway e DNS, evitando configurazioni manuali.

* **NAT (Network Address Translation)**: consente a più dispositivi di una rete privata di condividere un unico indirizzo IP pubblico per accedere a Internet, migliorando sicurezza e risparmio di indirizzi IP.
